글씨 쓰기 연습장 설명서

소개

이 연습장은 웹 기반으로 글씨 쓰기 연습지를 생성하고, 인쇄하거나 PDF로 저장할 수 있는 도구입니다.

핵심 기능

  • 직접 입력: 원하는 글자를 입력하여 연습지 생성(엔터로 생성 가능)
  • 짧은 글 랜덤: 미리 준비된 짧은 문장을 랜덤으로 추가(누를 수록 짧은 글 누적됨)
  • 긴 글 선택: 예문 목록에서 제목을 선택해 긴 글 불러오기
  • 이름 입력: 이름 입력하면 맨 위에 '날짜_이름' 생성
  • 글씨체 선택: 다양한 한글 폰트를 지원
  • 학년별 글자 크기: 1~6학년 기본 크기 제공
  • 글자 크기 조절: 12~72pt 사이로 화살표 사용하여 자유 변경 가능
  • 연습줄 수: 따라쓰기 칸과 빈 줄 수 설정
  • 인쇄 및 PDF 저장: 인쇄 미리보기 및 PDF 파일 저장
  • 설명서 열기: 메뉴의 "설명서" 버튼 클릭

사용 방법

  1. 페이지 접속: 브라우저에서 사이트를 열고 컨트롤 패널을 확인합니다.
  2. 글자 입력/선택: 연습할 글자에 직접 입력하거나, 긴 글 선택 드롭다운에서 예문을 선택합니다.
  3. 이름 입력/선택: 이름을 입력하면 연습장 맨 우측에 날짜와 이름이 기록됩니다.
  4. 설정 조정:
    • 글씨체: 원하는 폰트를 선택
    • 학년: 자동으로 적당한 글자 크기 적용
    • 크기(pt): 필요에 따라 수동 조정
    • 연습줄 수 및 따라쓰기 추가 옵션 설정
    • 세로/가로 인쇄 방향 선택
  5. 미리보기: 만들기 버튼 클릭 후 화면에 연습지가 표시됩니다.
  6. 인쇄: 인쇄 버튼을 누르면 인쇄 미리보기 후 출력
  7. PDF 저장: PDF 저장 클릭 시 자동 생성된 파일명(월일_이름.pdf)으로 다운로드/이름 미입력시(월일_연습장.pdf)
  8. 설명서 보기: 설명서 버튼 클릭 시 이 매뉴얼 페이지가 새 창으로 열립니다.

설정 저장

한 번 선택한 글씨체, 크기, 줄 수 등은 브라우저의 localStorage에 저장되어 다음 방문 시 자동 복원됩니다.

저장된 설정을 초기화하려면 브라우저 캐시 또는 로컬 스토리지를 삭제하세요.

자주 묻는 질문

Q: 예문 목록이 나오지 않아요.
A: examples.json이 서버에서 200 OK로 로드되는지 확인하고, 브라우저 캐시를 비활성화한 후 새로고침하세요.
Q: 줄이 페이지에서 분할돼요.
A: CSS의 page-break-inside: avoid가 적용되어 있습니다. 인쇄 방향과 여백 설정을 확인하세요.
